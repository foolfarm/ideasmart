# Analisi Layout Il Sole 24 Ore — Prima Pagina

## Struttura generale
- **Testata**: logo grande in alto a sinistra, dati mercati (FTSE MIB, spread) in una barra sotto
- **Layout**: griglia a 3 colonne principali + sidebar destra (più stretta)
- **Proporzioni**: colonne sinistra+centro = ~75% larghezza, sidebar destra = ~25%

## Gerarchia notizie
1. **Notizia principale (hero)**: titolo grande su 2-3 colonne, immagine, sommario
2. **Notizie secondarie**: titolo medio + 2-3 righe di testo, disposte in colonne
3. **Sidebar destra**: notizie brevi con titolo + 2 righe, divise per categoria (Panorama, Sfide Italia, ecc.)
4. **Notizie minori**: solo titolo + firma autore

## Tipografia
- Titolo hero: ~36-42px, serif bold
- Titoli secondari: ~22-28px, serif bold
- Body text: ~14-15px, serif, interlinea generosa
- Categorie/etichette: uppercase, piccolo, colore accent
- Firma autore: italic, piccolo

## Colori
- Sfondo: bianco carta (#FAFAF8)
- Testo: quasi nero (#1a1a1a)
- Accent: rosso/arancio per categorie urgenti
- Separatori: linee sottili grigio chiaro

## Sidebar destra
- Titolo sezione in uppercase bold
- Lista di notizie brevi con titolo + 2 righe testo
- Separatori orizzontali tra le notizie
- Eventuale pubblicità in fondo

## Elementi chiave da replicare per IdeaSmart
1. Barra ticker mercati (FTSE MIB → sostituire con dati AI/VC)
2. Notizia hero a tutta larghezza (editoriale del giorno)
3. Griglia 2+1 colonne per notizie secondarie
4. Sidebar destra con news stream (ultime notizie AI4Business + Startup)
5. Separatori di sezione con label uppercase
6. Font size più grandi: body min 15px, titoli secondari 20px+
